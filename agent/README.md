# Estrutura do agente

Esta pasta define a base para o agente de IA, separando configuração, prompts, memória, ferramentas e fluxos.

## Pastas
- `config/`: metadados e parâmetros do agente.
- `prompts/`: prompts base (system/developer/user).
- `memory/`: diretrizes para memória de curto e longo prazo.
- `tools/`: catálogo de ferramentas e integrações.
- `workflows/`: fluxos operacionais e playbooks.
